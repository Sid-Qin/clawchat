# clawchat

ClawChat channel plugin for [OpenClaw](https://github.com/openclaw/openclaw). Connects your OpenClaw agent to the ClawChat relay so iOS/Android apps can chat with it.

## Install

```bash
openclaw plugins install clawchat
```

Restart OpenClaw after installing. The plugin auto-generates a gateway token and prints a **QR code** to the terminal for pairing.

## Pair

Scan the QR code with the ClawChat iOS/Android app, or run:

```
/clawchat pair
```

to generate a new pairing code.

## Configuration (optional)

The plugin works out of the box. You can override defaults in `~/.openclaw/openclaw.json`:

```yaml
plugins:
  entries:
    clawchat:
      enabled: true
      config:
        token: <auto-generated>   # override if needed
        relay: wss://...           # defaults to hosted relay
        session: clawchat          # OpenClaw session key
```

## How it works

```
ClawChat App ──> ClawChat Relay <── OpenClaw Gateway (this plugin)
```

Both the app and the gateway connect **outbound** to the relay — no public IP or port forwarding needed.
